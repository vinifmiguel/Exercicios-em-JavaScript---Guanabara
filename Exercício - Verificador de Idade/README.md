# 👨‍🦳 Verificador de Idade Dinâmico

Segundo projeto prático desenvolvido durante o curso de **JavaScript** do canal Curso em Vídeo (Prof. Gustavo Guanabara). Este sistema valida o ano de nascimento, calcula a idade e exibe imagens dinamicamente conforme o gênero e faixa etária detectados.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação do formulário e interface.
* **CSS3**: Estilização moderna com sombras e centralização responsiva.
* **JavaScript (ES6)**: Lógica de cálculo, validação de dados e manipulação do DOM (Document Object Model).

## 🛠️ Funcionalidades
- [x] Validação de anos futuros ou campos vazios.
- [x] Identificação de Gênero (Masculino/Feminino).
- [x] Troca dinâmica de imagens usando `document.createElement('img')`.
- [x] Layout centralizado com `margin: auto` e `box-shadow`.
