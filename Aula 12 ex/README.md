# 🕒 Projeto: Hora do Dia (JavaScript - Exercício Aula 12)

Este projeto é o primeiro exercício prático do curso de JavaScript do Gustavo Guanabara (Curso em Vídeo). Ele foca na criação de uma interface dinâmica que interage com o usuário e o ambiente em tempo real.

## 🚀 Sobre o Desafio
O objetivo é construir um site que captura a hora atual do sistema e altera dinamicamente:
1.  **A mensagem exibida** (ex: "Agora são 15 horas").
2.  **A imagem de fundo** (Manhã, Tarde ou Noite).
3.  **A cor do background** do site para combinar com o período do dia.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica com `header`, `section` e `footer`.
* **CSS3:** Estilização moderna, incluindo sombras (`box-shadow`), bordas arredondadas (`border-radius`) e layouts centralizados.
* **JavaScript:** Manipulação do DOM e lógica baseada no objeto `Date`.

## 🧠 Aprendizados e Evolução
Durante o desenvolvimento deste projeto, foram aplicados conceitos de:
* Separação de responsabilidades em arquivos externos (.html, .css, .js).
* Tratamento e redimensionamento de imagens para otimização de layout.
* Centralização geométrica de elementos de bloco e inline.
