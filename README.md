# 💻 Exercícios em JavaScript - Curso em Vídeo

Bem-vindo ao meu repositório de evolução em JavaScript! Aqui organizo todos os desafios e projetos práticos realizados durante o curso do mestre Gustavo Guanabara, focando na transição para o desenvolvimento web profissional.

## 🛠️ Projetos em Destaque

Neste repositório, você encontrará soluções práticas que demonstram o domínio de diferentes camadas do desenvolvimento:

* **🕒 Projeto: Hora do Dia (Aula 12 ex):** Sistema dinâmico que altera imagem e cor de fundo em tempo real com base no relógio do sistema.
* **👤 Verificador de Idade (Em Desenvolvimento):** Interface interativa para validação de dados de usuário, cálculo de faixa etária e manipulação dinâmica de imagens.
* **🧪 Fundamentos (Aulas 06, 09, 10 e 12):** Exercícios focados em variáveis, operadores, estruturas condicionais e comandos básicos de interação.

## 🚀 Tecnologias Aplicadas
* **JavaScript (ES6+):** Lógica, manipulação do DOM e objetos globais.
* **HTML5:** Estruturação semântica de layouts.
* **CSS3:** Estilização moderna, responsividade e efeitos visuais.
